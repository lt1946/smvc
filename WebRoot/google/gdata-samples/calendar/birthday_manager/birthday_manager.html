Welcome to the birthday manager, an app that helps you keep track of your friends' birthdays. If you have a Facebook account and Greasemonkey installed, you can download this Greasemonkey script so that an 'ADD TO CALENDAR' button will appear next to birthdays on user profiles, and link to this page with the form filled in.
Because this is a third-party app that uses your Google account authentication, you'll need to grant access to it by clicking the "Login" button.
