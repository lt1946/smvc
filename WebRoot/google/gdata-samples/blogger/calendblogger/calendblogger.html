CalendBlogger

Introduction
CalendBlogger demonstrates the usage of combining both the Blogger JavaScript API and the Google Calendar Data JavaScript API. It allows the user to query his/her events on Google Calendar within a specific date range, and submit these events as a blog entry to blogger directly.

How it works
1. Login with AuthSub to your Google account for Calendar and Blogger.
2. Select a date range and all events within range will be displayed.
3. Edit the content of events by clicking on the display panel.
4. Click on "Blog >>>" to submit the entry to Blogger!
